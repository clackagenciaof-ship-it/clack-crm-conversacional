# Identidade visual — CLACK CRM Conversacional

## Direção visual atualizada

A identidade visual do Clack CRM Conversacional passa a seguir uma paleta verde turquesa, com aparência mais limpa, clara, profissional e institucional.

A proposta visual comunica:

- tecnologia;
- confiança;
- organização;
- clareza operacional;
- autoridade profissional;
- atendimento moderno;
- gestão comercial com leveza.

## Paleta principal

```text
Verde profundo: #005954
Verde institucional: #338b85
Turquesa principal: #5dc1b9
Turquesa claro: #9ce0db
Fundo gelo: #d5ffff
Branco: #ffffff
Cinzas de apoio: tons frios derivados de azul/cinza e verde acinzentado
```

## Uso recomendado

### Verde profundo — #005954

Usar em:

- sidebar;
- títulos principais;
- elementos de autoridade;
- áreas de maior contraste;
- texto institucional importante.

### Verde institucional — #338b85

Usar em:

- botões secundários;
- bordas ativas;
- indicadores comerciais;
- ícones;
- detalhes de cards.

### Turquesa principal — #5dc1b9

Usar em:

- gradientes;
- botões principais;
- barras de progresso;
- avatares;
- realces visuais.

### Turquesa claro — #9ce0db

Usar em:

- fundos suaves;
- badges;
- estados de hover;
- cards internos;
- divisórias suaves.

### Fundo gelo — #d5ffff

Usar em:

- fundo geral;
- áreas de respiro;
- login;
- páginas de leitura.

## Gradiente oficial

```css
linear-gradient(135deg, #005954 0%, #338b85 48%, #5dc1b9 100%)
```

## Aplicação na interface

A interface deve priorizar fundo claro, sidebar escura e cards brancos com bordas suaves.

Estrutura visual:

```text
Sidebar: #005954
Fundo geral: #d5ffff / tons claros derivados
Cards: branco translúcido
Botões principais: gradiente #005954 → #338b85 → #5dc1b9
Títulos: #005954
Textos secundários: verde acinzentado
Bordas: turquesa/cinza claro
```

## Observação

A paleta anterior em roxo, laranja, magenta e amarelo deixa de ser a paleta principal do produto. Ela pode ser reaproveitada futuramente em peças de marketing da Clack Growth Company, mas o CRM passa a usar a paleta verde turquesa como identidade principal do software.
